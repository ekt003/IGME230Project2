Made by Declan Behan and Emily Turner

Our Process and Credit Where Credit is Due

We started out with the basic pixel editor from our homework. The first big change made was adding RGB and hex color boxes in the tool box area. This allowed the user to manually select which color they wanted.

The second change was made to the overall aesthetic of the site. Minor tweaks to the css and html using flexbox left the site feeling better themed towards the topic.

The third major change was adding local saves to the site so that the user could save their work locally and return to it later. This was Declan's main job and it nearly killed him, not gonna lie. The way it works is that you enter in the name of your piece and hit 'Save Art'. It is then added to the load art menu. If you mess around with the canvas some more and decide to go back to the saved art, click the name on the drop down and click 'Load Art'

The fourth major change was hex to rgb and rgb to hex conversion. I actually ended up using Stack Overflow to help me with this problem.

Our final big change was the implementation of brush sizes and erasers. I created logos for each of the potential brushes and one for the eraser. Clicking on a different brush size changes the fillCell method so that it fills in the cells around it in addition to itself.

For a more in-depth look at our process, check out our GIT repository

The Requirements
